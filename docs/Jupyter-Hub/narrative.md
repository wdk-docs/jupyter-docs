# JupyterHub 叙事

## 描述

JupyterHub Narratives 为一组用户探索 Jupyter 笔记本的部署和扩展。
JupyterHub 允许灵活配置和部署，这使得 JupyterHub 对教育，行业研究团队和服务提供商有价值。
在这些叙述中，我们将研究部署，部署优势和最佳实践方面的差异。
